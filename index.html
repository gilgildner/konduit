How does Konduit work?

Konduit works by crowd-sourcing internet speed reporting by live users on real IPs. Because we all know that "high speed internet" sometimes means "we last upgraded our modem in 1998".

Lightweight Architecture

Konduit uses a lightweight MySQLite database hosted on EC2 for our API.

Privacy

We don't collect or sell any sensitive information.

Crowdsourced

We source data from anyone using our service on any web connection.

Platforms

Konduit is a web app, working on every desktop or mobile platform.
